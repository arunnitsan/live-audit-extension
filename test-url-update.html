🧪 URL Update Test - Accesstive Extension

Test Instructions

This page helps test the URL update functionality of the Accesstive extension:

  1. Load the extension in Chrome with this page open
  2. Open the sidebar by clicking the Accesstive extension icon
  3. Test URL updates using the methods below
  4. Check the console for detailed logging

Manual Testing

Test the URL update functionality manually:

1. Browser Console Commands

Open the browser console (F12) and run these commands:

// Check current status window.accesstiveSidebar.getStatus() // Test URL update window.accesstiveSidebar.testUrlUpdate('https://example.com') // Test with different URL window.accesstiveSidebar.testUrlUpdate('https://google.com')

2. Manual URL Change

Manually change the URL in the address bar to test real-world scenarios:

  • Go to https://google.com
  • Go to https://github.com
  • Go to https://stackoverflow.com

3. Programmatic Testing

Expected Behavior

When a URL change occurs, you should see:

  • ✅ Background script detects the URL change
  • ✅ Sidebar receives the notification
  • ✅ data-tab-url attribute gets updated on the script tag
  • ✅ Rescan is automatically triggered
  • ✅ Console logs show the complete flow
